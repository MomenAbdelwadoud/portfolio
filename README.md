# My Personal Portfolio

## View it from [here](https://momen.netlify.app)

## Tools used

1. NextJS
2. Tailwind CSS
3. Framer Motion
4. MUI
5. TypeScript

## How to run locally

1. Clone the repo.
2. Run `npm install` to get all the dependencies.
3. run `npx next dev` and go to `localhost:3000`.

## What I learned

* NextJS 13
